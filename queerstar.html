Queerstar

A progressive social media made by queers, for queers.
Queerstar is a part of an opening of interconnected decentralized radqueer sites (all of which broadly making up what we will refer to as the 'Progressiverse') made to give queers a safe space away from the conservative persecution faced on other platforms.

The Queerstar content policy terms of service is very simple: follow Consistent Progressivism, and don't be a conservative. There's nothing more and nothing less than that. :3

This principle of progressivism is applied consistently across the entire Progressiverse so that all queers on it can always be absolutely certain: if you're a queer, and you remain wholly dedicated to defending all of your other fellow queers, you'll never have your account attacked or be persecuted in any way.




What you'll need to join Progressiverse
1. The Tor Browser

One of the major features of Queerstar aside from it being exclusive to queers is that it's hosted over Tor routing as an onion site. This means that when you're using Queerstar, no one can see your Internet Protocol ('IP') address, which means that no one is able to know your location. This also adds the benefit of Progressiverse sites not needing to be hosted on domain registrars or website hosting services that can target Domain Name Systems ('DNS') to attack the sites and force them offline, similar to what happened to the now-defunct progressive Neocities site (which was moved to progressivism.lgbt).

All of this comes together to make every Progressiverse site a secure beacon of queerness in which queers are made completely immune from all conservative internet attacks. However, this feature comes with the effect that Progressiverse sites, due to using .onion Top-Level Domains ('TLDs'), will not load in any browser other than the Tor browser, because onion sites require the Tor protocol to render at all, which the clearnet has no access to due to Tor being decentralized instead of controlled by the Internet Corporation for Assigned Names and Numbers ("ICANN"), who controls the entirety of the clearnet.

Put simply, if you try to load an onion site in any browser other than the Tor browser, it won't work and you will get an error. This feature prevents clearnet access to onion sites so as to make sure that every queer that accesses them does so using the same protocol, thus defending sites and users from conservative internet attacks, since nothing is using the clearnet so as to be vulnerable to attack to begin with.


In order to get Queerstar (as well as any other Progressiverse site to load), download the Tor Browser, copy the onion site (which for Queerstar is ccxfwxo67dm65uwdxqeagmt7hzknvvmlu2si4vkgywt2kvicszjkz7qd.onion), and then paste the Uniform Resource Locator ('URL') of the onion site inside the Tor Browser.

2. An invite code from UnⒶpologetic Progressives (or a trusted queer)

Registration on all Progressiverse sites will require access to an invite code so as to make sure that only queers can get in. This vetting works to keep out conservatives so that everyone that joins Progressiverse actually follows progressivism, keeping the entire community as a supportive safe space for all identities.


The easiest way to get an invite code as of right now is to be fully verified in UnⒶpologetic Progressives, where you can get a code for free from the queers there. Alternatively, if you aren't in UnⒶpologetic Progressives, but you know a queer that already is or is already in Progressiverse, you can ask that queer to get you an invite code, and that will also work.
